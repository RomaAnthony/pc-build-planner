# 00_Hub

Fast working area for current orders, new captures, and active purchase research.

## Start Here

- `Current_Order/README.md` - current paid/pending order table.
- `Current_Order/REMAINING_PARTS.md` - what still needs a decision.
- `Current_Order/ROBERT_DELIVERY_SUMMARY.md` - clean summary for Robert/delivery coordination.
- `Research/` - broader article/review captures that support decisions.

## New Market Captures

Put every new unsorted market capture directly in `00_Hub/` first.

Use this when collecting from:

- Arukereso Hungary
- Ceneo Poland
- Hotline Ukraine
- store pages from any country
- screenshots or exported Markdown pages

Existing research captures that are already sorted live in:

- `Research/`

Do not sort while collecting. Drop files here, then Codex can move them into:

- `01_Market_Data/<date>/CPU/`
- `01_Market_Data/<date>/Motherboard/`
- `01_Market_Data/<date>/RAM/`
- `01_Market_Data/<date>/SSD/`
- `01_Market_Data/<date>/PSU/`
- `01_Market_Data/<date>/Cooler/`
- `01_Market_Data/<date>/Case/`
- `01_Market_Data/<date>/GPU/`
- `01_Market_Data/<date>/Fan/`

Good filename if you can choose it:

`2026-06-22 Site Country - Product.md`

Messy browser filenames are OK. The point of this folder is speed.
