# B850 vs X870 vs X870E — Chipset Deep Dive

For: Ryzen 9 9900X · MSI MAG B850 Tomahawk MAX WiFi

---

## Bottom Line

**B850 = X870 without mandatory USB4.** For this build, X870E's extra lanes are irrelevant (no dual GPU, no 4+ fast SSDs). B850 saves money with zero practical loss.

---

## Chipset Block Diagram (Simplified)

```
CPU (Ryzen 9 9900X)
├── 24 PCIe 5.0 lanes from CPU:
│   ├── 16 → GPU slot (always PCIe 5.0)
│   ├── 4 → Primary M.2 (PCIe 5.0)
│   └── 4 → Chipset uplink (PCIe 4.0 on B850, PCIe 5.0 on X870/X870E)
│
└── Chipset (Prom21)
    ├── B850: 4x PCIe 4.0 lanes available + SATA/USB
    ├── X870: 4x PCIe 4.0 lanes available + USB4 + SATA/USB  
    └── X870E: Dual Prom21 → 10x PCIe 4.0 lanes + USB4 + SATA/USB
```

---

## Lane Count Reality

For this build's planned components:

| Component | Lanes Used | B850 | X870 | X870E |
|---|---|---|---|---|---|
| RTX 5080 | 16 (CPU) | ✅ | ✅ | ✅ |
| KC3000 2TB M.2 | 4 (CPU, PCIe 5.0) | ✅ | ✅ | ✅ |
| Secondary M.2 or SATA | 4 (chipset) | ✅ | ✅ | ✅ |
| Chipset/IO overhead | ~4 (chipset) | ✅ | ✅ | ✅ |

**B850 has exactly enough lanes.** X870E's extra 6 lanes only help if you add:
- A third M.2 SSD (chipset PCIe 5.0)
- A capture card
- A 10G NIC

None of which apply to this build.

---

## USB4 — The Only Real X870 Advantage

X870 boards **must** include at least one USB4 40Gbps port. This means:
- One cable for external fast SSD (up to 3,000 MB/s)
- Future eGPU potential
- Video monitor daisy chaining over USB-C

For this build: USB4 is a convenience, not a need. The B850 has USB 3.2 Gen2 (10 Gbps) which is already fast for external drives.
